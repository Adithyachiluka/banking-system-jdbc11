# 🏦 Bank System Project

## 📌 About Project

This is a simple banking system developed using Java and MySQL.

---

## ⚙️ Technologies Used

* Java
* MySQL
* JDBC

---

## 🚀 How to Run Project

### 1️⃣ Setup Database

* Open MySQL
* Copy and run the file `database/bankdb.sql`

### 2️⃣ Run Java Program

* Open terminal in `src` folder
* Compile:
  javac BankSystem.java
* Run:
  java BankSystem
